# 🛒 E-Commerce Web App

A modern and responsive E-Commerce Web Application built using **HTML**, **CSS**, and **JavaScript**.

This project simulates a basic online shopping experience with product listings, shopping cart functionality, and a responsive user interface. Backend development and database integration will be added in future updates.

---

## 🚀 Features

✔️ Responsive Design
✔️ Product Listing Page
✔️ Search Products
✔️ Add to Cart
✔️ Remove Items from Cart
✔️ Dynamic Cart Total Calculation
✔️ Local Storage Support
