# 🛒 E-Commerce Web App

An E-Commerce Web Application designed to provide a seamless online shopping experience. The platform allows users to explore products, manage their shopping carts, and place orders through an intuitive and responsive interface.

## ✨ Features

### 👤 User Features

* User Registration and Login
* Secure Authentication
* Browse Products by Categories
* Search and Filter Products
* View Product Details
* Add and Remove Items from Cart
* Update Product Quantity in Cart
* Place Orders
* View Order History
* Responsive User Interface

### 🔐 Admin Features

* Admin Dashboard
* Add, Update, and Delete Products
* Manage Product Categories
* Manage Orders
* Manage Users

## 🛠️ Tech Stack

**Frontend**

* HTML5
* CSS3
* JavaScript
* React.js
* Bootstrap/Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB
