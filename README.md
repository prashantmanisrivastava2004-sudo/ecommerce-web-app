E-Commerce Web App

A responsive and modern E-Commerce Web Application built using HTML, CSS, and JavaScript. This project focuses on creating a clean user interface and smooth user experience for online shopping. Backend integration and database support will be added in future updates.

🚀 Features
Responsive Design
Product Listing Page
Product Search Functionality
Add to Cart System
Cart Item Management
Dynamic Price Calculation
Modern UI/UX
Mobile-Friendly Layout
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
